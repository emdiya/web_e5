Web Server Side Technologies (Back-end Web Development) Client Server Processing មិនថាវាយអាសយដ្ឋាន(URL) ចុចតំណ(link) ឬចុចប៊ូតុងលើសំណុំបែបបទ(form) ធម្មតាweb browser ស្នើទៅ web server ប្រើ HTTP (Hyper Text Transfer Protocol) ការស្នើរួមមាន៖ -URL ដែលបញ្ជាក់អាសយដ្ឋានមេម៉ាស៊ីននិងធនធាន(វេបផេក ឧបករណ៍ត្រូវដើរ ។ល។) -សកម្មភាពចាំបាច់(ទទួលឯកសារ ផ្ញើទិន្នន័យ។ល។) -ព័ត៌មានបន្ថែមបើមាន(ទិន្នន័យដែលត្រូវការ) -Client Side Cookies (ផ្ទុកទិន្នន័យដែលប្រើវេបសាយម្ដងៗ) web server តបទៅលើ web browserវិញជាសារជោគជ័យ(200) មិនមាន(404) ឬមិនមានការអនុញ្ញាត(403) web browser ត្រូវបានវេបផេកដែលserver បានផ្ដល់ឲ្យចំពោះការស្ទើរដែលជោគជ័យ។
